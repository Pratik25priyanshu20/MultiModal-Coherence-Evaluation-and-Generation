# Experimental Modules

These modules are not part of the main pipeline or reported experiments.
They are kept for reference and potential future development.

## training/
Contrastive learning for embedding alignment (InfoNCE loss, trainable projection heads).
Status: Not integrated — the main pipeline uses pre-trained CLIP/CLAP alignment.

## narrative/
Multi-beat story generation and story structure schema.
Status: Exploratory — not used in any experiment.
